# Notepad App with PyQt6

A simple desktop Notepad application developed using **Python** and **PyQt6**. The application provides basic text editing functionality similar to Windows Notepad, allowing users to create, open, edit, and save text files through an intuitive graphical interface.

---

## Features

* Create a new text file
* Open existing text files
* Save files
* Save files with a new name (Save As)
* Undo and Redo support
* Cut, Copy, and Paste functionality
* Clean and user-friendly GUI
* Menu bar with File and Edit options

---

## Technologies Used

* Python 3
* PyQt6

---

## How to Use

1. Launch the application.
2. Use the **File** menu to:

   * Create a new file
   * Open an existing file
   * Save the current file
   * Save As to create a new file
3. Use the **Edit** menu to:

   * Undo
   * Redo
   * Cut
   * Copy
   * Paste
